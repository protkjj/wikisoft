================================================================================
확정급여채무평가 데이터 검증 AI Agent - 사용방법
================================================================================

1단계: 라이브러리 설치 (처음 한 번만)
---------------------------------------
명령 프롬프트(CMD) 또는 PowerShell을 열고 프로젝트 폴더로 이동한 후:

    pip install -r requirements.txt

실행하세요. 필요한 라이브러리가 자동으로 설치됩니다.


2단계: API 키 설정
---------------------------------------
프로젝트 폴더에 ".env" 파일을 생성하고 다음 내용을 입력하세요:

    OPENAI_API_KEY=여기에_당신의_OpenAI_API_키_입력

예시:
    OPENAI_API_KEY=sk-1234567890abcdef...

※ OpenAI API 키는 https://platform.openai.com/ 에서 발급받을 수 있습니다.


3단계: 엑셀 파일 준비
---------------------------------------
검증할 엑셀 파일을 "data" 폴더에 넣으세요.

예시:
    data/푸본현대생명_2212_확정급여채무평가_작성요청자료.xlsx


4단계: 프로그램 실행
---------------------------------------
명령 프롬프트(CMD) 또는 PowerShell에서:

    python main.py

실행하세요.


5단계: 결과 확인
---------------------------------------
검증이 완료되면 "output" 폴더에 결과 파일이 생성됩니다.

파일명 형식: 감사_의견서_YYYYMMDD_HHMMSS.xlsx

예시:
    output/감사_의견서_20251230_221222.xlsx

엑셀로 열어서 검증 결과를 확인하세요.


================================================================================
전체 실행 순서 요약
================================================================================

1. pip install -r requirements.txt  (처음 한 번만)
2. .env 파일 만들고 API 키 입력
3. 엑셀 파일을 data/ 폴더에 넣기
4. python main.py 실행
5. output/ 폴더에서 결과 확인


================================================================================
주의사항
================================================================================

- .env 파일은 반드시 프로젝트 폴더 최상위에 있어야 합니다.
- 엑셀 파일은 반드시 data/ 폴더에 넣어야 합니다.
- OpenAI API 키가 유효하고 충분한 크레딧이 있어야 합니다.
- 데이터가 많으면 검증에 시간이 걸릴 수 있습니다.


================================================================================
문제 해결
================================================================================

오류: "OPENAI_API_KEY가 설정되지 않았습니다"
→ .env 파일이 있는지, API 키가 올바르게 입력되었는지 확인하세요.

오류: "엑셀 파일을 찾을 수 없습니다"
→ data/ 폴더에 엑셀 파일(.xlsx 또는 .xls)이 있는지 확인하세요.

오류: "모듈을 찾을 수 없습니다"
→ pip install -r requirements.txt를 다시 실행하세요.
